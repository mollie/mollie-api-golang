# MandateMethod

Payment method of the mandate.

SEPA Direct Debit and PayPal mandates can be created directly.


## Values

| Name                       | Value                      |
| -------------------------- | -------------------------- |
| `MandateMethodCreditcard`  | creditcard                 |
| `MandateMethodDirectdebit` | directdebit                |
| `MandateMethodPaypal`      | paypal                     |
