# GetMandateMethod

Payment method of the mandate.

SEPA Direct Debit and PayPal mandates can be created directly.


## Values

| Name                          | Value                         |
| ----------------------------- | ----------------------------- |
| `GetMandateMethodCreditcard`  | creditcard                    |
| `GetMandateMethodDirectdebit` | directdebit                   |
| `GetMandateMethodPaypal`      | paypal                        |
